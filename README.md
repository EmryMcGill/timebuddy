# Time Buddy
Author: Emry McGill

## Description:
A pomodoro stle timer website to track how long you spend working on different things. 

Features:
- Add projects
- Adjust length of break
- view overview of time logs plotted on a bar chart

## Stack:

Frontend: HTML, CSS, JS

Backend: Pocketbase, Nginx

 ## Seeking Feedback!
 
This is my first app using Pocketbase as a database and for authentication, so I would appriciate any feedback on my implimentation, and if there is anything I can do to improve it.
